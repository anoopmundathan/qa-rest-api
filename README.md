# qa-rest-api
Treehouse course - Build REST API with Express

## Getting Started
### Setup
```
$ git clone https://github.com/anoopmundathan/qa-rest-api.git
$ cd qa-rest-api
```
### Install
```
$ npm i
```
### Run API Server
```
npm start
```

You may consume this API using [clien-app](https://github.com/anoopmundathan/client_app.git)
